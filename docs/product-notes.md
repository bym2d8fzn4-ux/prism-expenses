# Product Notes

## Shared Direction

The web app and iOS app should feel like the same product:

- Product name: Expenses
- Brand: PrismJet
- Primary color: Prism orange
- Primary views: Expenses and Incentives
- Statuses: Not Submitted, Submitted, Paid
- Grouping options: Trip, Week, Airport, Aircraft, Vendor, Category

## Platform Notes

The web app remains static so it can publish through GitHub Pages without a server bill. Data is stored locally in the browser with import/export backup support.

The iOS app is native SwiftUI. Data is stored on-device with manual backup import/export and receipt image storage.
